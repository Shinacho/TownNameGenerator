

DQ風　地名ジェネレータ	1.0


１．ツールについて
    伝説の勇者の子孫が訪れそうな町の名前を生成するツールです。
　　所謂、「よく使う20文字のカタカナ」を、適当に組み合わせて出力しています。
　　動作にはJavaランタイムが必要。

２．使い方
    1 起動して、長さを選んで生成ボタンを押せばOK！生成された名前をクリックするとクリップボードに保存できます。
　　2 一度にたくさん出力した場合は「まとめて生成」タブからファイルに保存できます。


３．データ仕様
　　「まとめて生成」モードでは、テキストファイルに1行1件で出力します。    

４．既知の不具合
　　たぶん特にない。
